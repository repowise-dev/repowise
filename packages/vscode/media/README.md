# media

Static assets referenced by the extension manifest.

- `repowise.svg` is the activity-bar icon. The editor applies container icons
  as an alpha mask and themes the color itself, so the brand mark's alpha
  channel is the shape that renders.
- `icon.png` is the Marketplace listing icon (256x256), referenced from the
  top-level `icon` field in `package.json`.
- `walkthrough/` holds the getting-started step bodies.
